HTML5 and CSS3 Notes

About Meta tags

Meta tags are useful for storing information that search engines rely on to find and display a webpage within the results. Search engines that rely on meta tags include Google, Duck Duck Go, and Bing

Meta Tag types can best be remembered by the acronym D.A.C.K.

Description- the blurb under the website title that gives you a preview of site content
Author-the name of who authored the website
Charset- I'm not sure what this is yet, but it usually is set to UTF-8
Keywords- This is how the search engine determines which sites are relevant to the search term.

CSS New Terms

There are some new terms in CSS that I have not previously learned. They will be defined in the list below in the order that I learned about them.

  1. em
    This allows the text to be rendered more flexibly. If a user sets their computer defaults to magnify or shrink all the text on their screen, this setting will allow the browser to follow this rule. Otherwise, the text will always be the same size (by using px) no matter what. 1em equals the default size of the element. 2em will double the font-size, and so on.

  2. Justified Alignment
    This makes the margins on both the left and the right side of the paragraphs even and flush by changing the spacing between words. It creates a nice, even look on webpages

  3. Font-Family Rule
    It is recommended to use 3 fonts for each specification. The most unique and preferred to the most generic that every computer and browser supports, just in case. A good recommendation is to make the 3rd "fallback" option either serif (fancy) or sans-serif (easier to read)
